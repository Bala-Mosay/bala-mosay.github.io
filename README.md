# Bala Mosay J – Portfolio

My personal portfolio website built with [Astro](https://astro.build).

Showcasing my work in AI/ML, deep learning, and full-stack development.

---

## 🚀 Getting Started

To run the project locally:

```bash
# 1. Clone this repo
git clone https://github.com/Bala-Mosay/BalaMosay.github.io.git
cd BalaMosay.github.io

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
```
